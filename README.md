# mu8429's Blog

个人博客，使用 VitePress 构建。

## 技术栈

- [VitePress](https://vitepress.dev/) - 静态站点生成器
- [Vue 3](https://vuejs.org/) - 前端框架
- [GitHub Pages](https://pages.github.com/) - 托管服务

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages。当推送代码到 `main` 分支时，会自动触发构建和部署。
